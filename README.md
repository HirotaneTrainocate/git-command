## calculate.py について
`../calculate.py` には、Pythonで四則演算を行う関数が定義されています。

- `add(a, b)`: 足し算を行います
- `subtract(a, b)`: 引き算を行います
- `multiply(a, b)`: 掛け算を行います
- `divide(a, b)`: 割り算を行います（`b` が 0 の場合は `ValueError` を送出します）
